VB RAW Viewer

By Simon Price

Important Note!

Due to a bug in VB, don't run this program from the VB IDE because (in my experience) half the polygons don't show up! Run the compiled .exe to see the correct results.

About

This program is very simple, you can view .raw files and whilst the object loaded spins round you can zoom in and out. It does not use DirectX or OpenGL to do this, yet it still runs (fairly) fast because of usage of Windows API and (fairly) good coding. It could run faster but this was just a quick project to help me understand the .raw file format because I am making a 3D modelling program with import/export options.
